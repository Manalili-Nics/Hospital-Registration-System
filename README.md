# 🏥 SafeCare Health Hospital - Complete Healthcare Management System

## 📋 Project Overview
A **complete healthcare registration and appointment management system** built with Bootstrap 5 and JavaScript. This web application allows patients to register, book appointments with specialist doctors, manage appointments, and leave reviews.

## 🎯 Activity Requirements Met

### ✅ Bootstrap Components
- Navbar (Sticky navigation)
- Buttons (Primary, Success, Danger, Outline)
- Cards (Doctor profiles, Stats card)
- Form (Patient registration with validation)
- Tables (Upcoming/Completed/Cancelled appointments)

### ✅ Interactive Components
- **Carousel** - Hospital facility showcase
- **Modal** - Hospital guidelines & review system
- **Accordion** - FAQ section
- **Alerts** - Success/Error notifications
- **Badges** - Status indicators, ratings

### ✅ JavaScript Integration
- Dynamic table population
- Form validation
- Modal show/hide events
- Event listeners (show.bs.modal)
- Real-time UI updates

## 🚀 Features

### Patient Registration
- Full name, email, mobile number
- Date of birth, blood type
- Emergency contact information
- Form validation for email and phone numbers

### Appointment Management
- Book appointments with any doctor
- **Complete** appointment (moves to Completed tab)
- **Cancel** appointment (moves to Cancelled tab)
- View appointments by status (Upcoming/Completed/Cancelled)

### Review System
- Rate doctors (1-5 stars)
- Write detailed reviews
- View all patient reviews for each doctor

## 💻 Technologies Used

- **HTML5** - Structure
- **CSS3** - Custom styling
- **Bootstrap 5** - UI components & responsiveness
- **JavaScript** - Dynamic functionality

## 📱 Responsive Design
- Fully responsive on desktop, tablet, and mobile devices
- Bootstrap grid system ensures proper layout on all screen sizes

## 🎮 How to Use

1. **Register** - Fill out the patient registration form
2. **Book** - Browse doctors and click "Book Appointment"
3. **Manage** - View your appointments in the tabs
4. **Complete** - Mark appointments as completed after visit
5. **Review** - Leave ratings and feedback for doctors
